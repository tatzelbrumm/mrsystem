ByteStream

Dieses Modul stellt Funktionnen zur Verfügung, um aus einen Strom von Bytes Datentypen wie short und long zu lesen und zu schreiben. Die Bytes werden in per Motorola Reihenfolge (big endian) erwartet. Sie können auch auf einer ungeraden Adresse liegen. Die Bytereihenfolge kann auf Intel (little endian) umgestellt werden. Der Einsatz des Moduls vermeidet ein "darüberlegen" einer Struktur, bei der ein bestimmtes Alignement erzwungen werden müßte.

Sprachen

C, C++

Includefiles

#include <bytestream.h>

Librays

libbytestream.a

Funktionen

Lesen aus dem Bytestrom

unsigned long GetLongFromByteArray(unsigned char *Data);
unsigned int GetIntFromByteArray(unsigned char *Data);

Schreiben in einen Bytestrom

void SetLongToByteArray(unsigned char *Data, unsigned long Value);
void SetIntToByteArray(unsigned char *Data, unsigned int Value);

Umstellen der Endianess

#define BYTESTREAM_LITTLE_ENDIAN 1
#define BYTESTREAM_BIG_ENDIAN    2

void ByteStreamSetEndianess(int Endianess);
