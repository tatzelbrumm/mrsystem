Die Konfigurationsdatei /etc/mrlok

Die Datei /etc/mrlok beschreibt, welche Aktionen für eine spezielle Lok ausgeführt werden sollen.

Die erste Zeile enthält die Adresse der Lok und durch ein Komma getrennt den Loknamen.

Jede weiter Zeile der Datei enthält den Namen der Funktionstaste und durch ein Komma getrennt die auszuführende Aktion. Ein Beispiel sieht wie folgt aus: 

1,LOKLISTE
F1,POSIXLY_CORRECT=1 /usr/local/bin/mrinjectipc 25 -1 -1 0 0 1 0 0 0 0
F2,sudo /etc/init.d/mrsystem restart
F3,sudo /sbin/reboot -f
F4,sudo /sbin/halt -f
