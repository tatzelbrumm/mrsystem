Konfiguration

Die Seite mit dem Menü für die Konfiguration enthält Einträge für die Konfiguration der mrsystem Software, des Status der Geräte am CAN Bus, der Systemkonfiguration, ..., das Eoinspielen von Updates und Neustart des mrsystem, des Geräts und dem Shutdown des Geräts.

mrsystem Konfiguration

Der Tab MR-System erlaubt es das Verhalten der Software festzulegen. Das Formular ist damit ein Editor für die /etc/mrsystem Datei.

Der Tab Start erlaubt es festzulegen, welche Komponenten der Software gestartet werden sollen. Damit wird das Verhalten bestimmt und die Software an die vorhandenen Schnittstellen angepaßt. Das Formular ist damit ein Editor für die /etc/mrstart Datei.

Der Tab Spezial-Lok erlaubt es festzulegen, welche Aktionen ausgeführt werden sollen, wenn eine Funktion einer Lok mit vorgegebener Adresse und Namen ausgeführt werden sollen. Das Formular ist damit ein Editor für die /etc/mrlok Datei.

CAN Geräte / Messwerte / Konfiguration

Die Seite mit dem Menü für die CAN Geräte enthält Einträge für eine Übersicht der Geräte, deren Messwerte und die Konfiguration der Geräte (soweit vorhanden).

CAN Geräte

Dieser Tab zeigt als Beispiel ein MS2 am CAN Bus.

Dieser Tab zeigt als Beispiel die Gleisbox am CAN Bus.

Messwerte der CAN Geräte

Dieser Tab zeigt als Beispiel die Messwerte der Gleisbox.
